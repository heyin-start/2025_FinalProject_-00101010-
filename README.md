# 2025_FinalProject_-00101010-
Final Project for Group 00101010
In the project, we view average session length as user's average time spent in the task, session counts as intensity user's will to participate in the task again, and in app purchase amount as indication of user's will to good performance in a task.
We will determine if there exist game genres such that the average session length, session counts, and in game purchase amount are significantly higher that all other game genres. We will also determine if there exist correlation between average session length and session counts, if there exist correlation between session counts and in game purchase amount, and if there exist correlation between average session length and in game purchase amount.
